### 3.1 新增模块类 ([block.py](ultralytics/nn/modules/block.py))

**DSBottleneck** — 深度可分离瓶颈块：
```python
class DSBottleneck(nn.Module):
    # depthwise conv (groups=c1) + pointwise conv (1x1)
    # 参数量比标准 Bottleneck 减少 87.2%
    def __init__(self, c1, c2, shortcut=True, g=1, k=(3,3), e=0.5):
        # cv1: DW(c1→c1) + PW(c1→c_)
        # cv2: DW(c_→c_) + PW(c_→c2)
```

**C3k2Lite** — 轻量 C3k2 变体：
```python
class C3k2Lite(C2f):
    # 用 DSBottleneck 替换内部 Bottleneck
    # Drop-in replacement for C3k2
    def __init__(self, c1, c2, n=1, ...):
        self.m = nn.ModuleList(DSBottleneck(...) for _ in range(n))
```

**P2Proxy** — P2 代理细节支路：
```python
class P2Proxy(nn.Module):
    # CSP 风格：1x1 Conv → n×Bottleneck → Concat → 1x1 Conv
    # 从 P2/4 引出，保留细粒度纹理
    def __init__(self, c1, c2, n=1):
        self.cv1 = Conv(c1, c_, 1, 1)
        self.m = nn.ModuleList(Bottleneck(c_, c_) for _ in range(n))
        self.cv2 = Conv((1+n)*c_, c2, 1)
```

**ProxyFuse** — 抗混叠语义回灌融合：
```python
class ProxyFuse(nn.Module):
    # stride-2 Conv 降采样 P2 → 通道对齐 → P3 全局语义门控
    def __init__(self, c_p2, c_p3, c2=None, gate_ratio=4):
        self.downsample = nn.Sequential(DW3x3_s2, BN, SiLU)
        self.align = Conv(c_p2, c2, 1, 1)      # P2→target
        self.reduce = Conv(c_p3, c2, 1, 1)      # P3→target
        self.gate = nn.Sequential(GAP, 1x1Conv, SiLU, 1x1Conv, Sigmoid)
    def forward(self, x):  # x = [p2, p3]
        return self.reduce(p3) + self.align(self.downsample(p2)) * self.gate(p3)
```

**NDA** — 近邻解耦适配器：
```python
class NDA(nn.Module):
    # 局部均值池化 → 差分 → 1x1 Conv
    def __init__(self, c1, c2, kernel_size=5):
        self.pool = nn.AvgPool2d(kernel_size, stride=1, padding=kernel_size//2)
        self.cv1 = Conv(c1, c2, 1, 1)
    def forward(self, x):
        return self.cv1(x - self.pool(x))  # 局部对比增强
```
